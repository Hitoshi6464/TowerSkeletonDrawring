# TowerSkeletonDrawring
This program draws the skeleton of a power transmission tower in Google Earth. 
The program was created using VB.net or C#.net in Visual Studio 2022.

1.TowerSkeletonData.xlsm - The TowerSkeletonData.xlsm file is skeleton data for a transmission tower.
2. WinFormsApp3 mod4     - WinFormsApp3 mod4 is a program that draws steel towers by inputting longitude
                           and latitude and ground elevation on Google Earth.   
3. WinFormsApp4 mod2     - WinFormsApp4 mod2 is a program that draws G-lines and power lines between towers 
                           drawn on Google Earth.
4. WinFormsApp21 mod     - The WinFormsApp21 mod is a program that displays the ground at any longitude and
                           latitude position on Google Earth using a mesh with a pitch of 5 or 10 meters.
